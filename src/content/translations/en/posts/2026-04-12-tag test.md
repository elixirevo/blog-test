# Elixir Evo Journal

This is a SvelteKit-based static blog template. It is configured to prerender the entire site using `@sveltejs/adapter-static`, deploy to GitHub Pages via GitHub Actions, and allow editing of Markdown content within the Pages CMS.

## Stack

- SvelteKit + `@sveltejs/adapter-static`
- GitHub Pages deployment via `.github/workflows/deploy.yml`
- Pages CMS via root `.pages.yml`
- Markdown posts in `src/content/posts`
- DeepL translation sync into `src/content/translations/<locale>/posts`

## Local development

```sh
bun install
bun run dev
```

The `Pagefind` search index is generated after a static build. To verify the search functionality, follow the steps below.

```sh
bun run build
bun run preview
```

To update registered translation language files locally, enter your DeepL API key and run the command below.

```sh
DEEPL_API_KEY=your-key bun run translate:posts
```

## Content structure

- `src/content/site.json`: Meta information such as site title, description, and URL
- `src/content/posts/*.md`: Frontmatter + Markdown body
- `src/content/translations/<locale>/posts/*.md`: Language-specific translations generated and committed by DeepL
- `static/uploads`: Images uploaded via the CMS

You can add multiple tags to a post’s frontmatter in addition to categories. Tags are saved without `#` and displayed on the screen as `dev`.

In Pages CMS, you first create tags in the `Tags` collection, then select multiple tags from the `Tags` field on the post editing screen.

```yaml
category: Technical
tags:
  - dev
  - git
```

Single-line strings are also supported.

```yaml
tags: 'dev, git'
```
