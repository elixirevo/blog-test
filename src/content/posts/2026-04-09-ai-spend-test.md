# AI 지출 코치 서비스 기획서

가제: **머니리크 코치(Money Leak Coach)**

## 1. 서비스 개요

**한 줄 정의**  
카드, 구독, 지출 데이터를 분석해 **반복 지출 패턴을 찾고**, **왜 돈이 새는지 해석하고**, **바로 실행할 행동 가이드와 알림**까지 제공하는 LLM 기반 소비 코칭 서비스.

**서비스 핵심 가치**  
기존 가계부가 “얼마를 썼는지” 보여주는 데 그쳤다면, 이 서비스는 “**왜 이렇게 쓰게 되었는지**”와 “**지금 무엇을 바꿔야 하는지**”까지 알려준다.  
즉, 단순 기록형 서비스가 아니라 **행동 변화형 금융 어시스턴트**를 지향한다.

---

## 2. 문제 정의

사용자는 카드 명세서와 소비 내역은 볼 수 있지만, 실제로는 아래 문제를 자주 겪는다.

- 구독이 많아졌는데 무엇이 자동결제 중인지 한눈에 파악이 어렵다.
- 커피, 배달, 이동, 쇼핑처럼 **소액 반복 지출**이 누적되어도 체감이 잘 안 된다.
- “이번 달 왜 돈이 부족하지?”라는 질문에 대해 숫자는 보여도 **원인 해석**이 안 된다.
- 목표 예산을 세워도 중간에 이탈하는데, 이를 미리 잡아주는 **경고/피드백 시스템**이 부족하다.

이 서비스는 이 문제를 해결하기 위해, 사용자의 소비를 단순 분류가 아니라 **패턴·원인·행동** 단위로 재구성한다.

---

## 3. 서비스 목표

이 서비스의 목표는 4가지다.

1. **반복 지출 패턴 자동 탐지**
정기결제, 소액 반복, 특정 시간대/요일 편중 지출, 특정 카테고리 급증 등을 자동으로 잡아낸다.

2. **지출 원인 해석**
  “이번 달 외식비가 늘었다” 수준이 아니라  
   “야근이 많아지면서 평일 밤 배달 소비가 증가했고, 주말 외식까지 겹치며 지출이 누적됐다”처럼 맥락을 설명한다.

3. **행동 가이드 제시**
  “줄이세요”가 아니라  
   “이 구독은 해지 후보”, “배달 예산 상한 설정”, “주 2회 무지출 챌린지”, “현재 사용 카드 혜택 미스매치”처럼 실행 가능한 액션으로 바꿔준다.

4. **실시간 경고와 진행 피드백 제공**
  과소비 조짐이 보이면 미리 경고하고, 목표 대비 현재 진행률을 이해하기 쉽게 피드백한다.

## 4. 타깃 사용자

### 1차 타깃

20~40대 직장인 / 프리랜서 / 1인 가구

### 공통 특성

- 카드 2장 이상, 구독 서비스 여러 개 사용
- 월급은 들어오는데 “왜 항상 부족한지” 감이 잘 안 잡힘
- 가계부를 꾸준히 쓰기 어렵고, 대신 **자동 분석**을 원함
- 숫자보다 **해석과 조언**을 더 필요로 함

### 대표 페르소나

“OTT, 음악, 클라우드, 커피, 배달, 택시, 쇼핑이 조금씩 계속 나가는데,  
정작 어디서 돈이 새는지 정확히 모르고 월말마다 불안한 사용자”

## 5. 핵심 기능

## 5-1. 반복 지출 패턴 탐지

탐지 대상 예시는 다음과 같다.

- **정기 구독 패턴**: OTT, 음악, 생산성 툴, 클라우드, 멤버십 등
- **숨은 반복 지출**: 매주 비슷한 요일/시간대에 발생하는 소비
- **소액 누수 패턴**: 커피, 편의점, 배달, 택시 등 자잘하지만 빈번한 지출
- **이상 증감 패턴**: 특정 카테고리 소비가 평소보다 급증
- **중복 결제/비효율 구독**: 유사 서비스 동시 결제, 거의 사용하지 않는 구독
- **습관성 소비 패턴**: 야간 소비, 스트레스성 쇼핑, 월급일 직후 과소비

### 기능 결과 예시

- “최근 8주간 금요일 밤 배달 지출이 반복적으로 발생”
- “최근 3개월 평균 대비 카페 지출 42% 증가”
- “유사한 영상 구독 3개를 동시에 유지 중”
- “월초 5일 내 소비가 전체 예산의 37%를 차지”
