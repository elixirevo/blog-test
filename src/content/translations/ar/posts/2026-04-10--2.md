# مجلة Elixir

قالب مدونة ثابتة مبني على SvelteKit. تم تكوينه بحيث يتم عرض الموقع بالكامل مسبقًا باستخدام `@sveltejs/adapter-static`، ونشره على GitHub Pages باستخدام GitHub Actions، مع إمكانية تحرير محتوى Markdown في نظام إدارة المحتوى (CMS) الخاص بـ Pages.

## Stack

- SvelteKit + `@sveltejs/adapter-static`
- نشر GitHub Pages عبر `.github/workflows/deploy.yml`
- Pages CMS عبر الجذر `.pages.yml`
- منشورات Markdown في `src/content/posts`
- مزامنة ترجمة DeepL مع `src/content/translations/en/posts`

## التطوير المحلي

```sh
bun install
bun run dev
```

يتم إنشاء فهرس البحث `Pagefind` بعد البناء الثابت. لاستخدام وظيفة البحث، اتبع الخطوات التالية.

```sh
bun run build
bun run preview
```

لتحديث ملفات الترجمة الإنجليزية محليًا، أدخل مفتاح DeepL API وقم بتنفيذ الأمر التالي.
