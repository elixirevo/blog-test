# Elixir Journal

SvelteKit ベースの静的ブログテンプレートです。`@sveltejs/adapter-static` を使用してサイト全体をプリレンダリングし、GitHub Actions を通じて GitHub Pages にデプロイし、Pages CMS で Markdown コンテンツを編集できるように構成されています。

## Stack

- SvelteKit + `@sveltejs/adapter-static`
- `.github/workflows/deploy.yml` 経由の GitHub Pages デプロイ
- ルートディレクトリにある`.pages.yml`経由のPages CMS
- `src/content/posts` 内の Markdown 投稿
- DeepLの翻訳を`src/content/translations/en/posts`に同期

## ローカル開発

```sh
bun install
bun run dev
```

`Pagefind` の検索インデックスは、静的ビルド後に生成されます。検索機能まで確認するには、以下の手順を実行してください。

```sh
bun run build
bun run preview
```
